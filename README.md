## Lodash优化(lodash-webpack-plugin、 babel-plugin-lodash)

都是列出Gzip后的大小

**未引入lodash前**：

app.js：827KB
vendors.js：692KB

**引入lodash后，优化前**：

app.js：1.1MB
vendors.js：692KB

**引入lodash后，优化后**：

app.js：829KB
vendors.js：692KB
